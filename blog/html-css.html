What are the best practices associated with using
classes vs. ids?


What are classes and Id's?

image.png Classes and id's are attributes used in HTML and CSS with the purpose of styling web pages

How do you apply classes and Id's?

At first it can be complicated to understand which one you should use as they work in a very similar way
A class can be used to select a group of elements, like multiple paragraphs that could share the same style
An ID can only select a single element and cannot be used to style multiple elements

What are the best practices?

There are no rules for which you should use one rather than the other as both Id's and classes have a similar purpose but with time
chosing one over the other will leave you with cleaner code.
